## Kurulum

### Clone

```bash
git clone CLONE_URL
```

### Proje klasörün girin

```bash
cd Flask-Todo-App
```

### Gerekli paket ve modülerin kurulumu

```bash
pip install -r requirements.txt
```

### Projeyi çalıştırın

#### Windows

```bash
python main.py
```

#### Linux

```bash
python3 main.py
```
